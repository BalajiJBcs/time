## Customize options

- `fg`: Font color. Examples:
  - `?fg=red`
  - `?fg=#ff00ff`
- `bg`: Background color. Examples:
- `font`: Font family. Examples:
  - `?font=monospace`
- `seconds`: Show seconds if this argument present.
